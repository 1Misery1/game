# Interactive Solar System (Unity)

This repository contains a **new, standalone Unity project** for an interactive solar system assignment.

It is **independent from any 2D game project** and is not a continuation, branch, or module of a previous 2D game.

## Project Scope

- Interactive solar system scene for kids
- Clickable celestial bodies (planets/moon)
- Camera focus and return-to-main-view behavior
- Basic orbit and rotation motion
- Planet texture-based visuals
